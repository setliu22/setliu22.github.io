# Seton Liu Portfolio

A static personal portfolio inspired by the minimal structure of a garment label:
condensed type, clean rules, project codes, and quiet editorial spacing.

## Edit Content

- Update page copy and project cards in `index.html`.
- Adjust colors and layout in `styles.css`.
- Replace `assets/monogram.svg` if you want a different mark.

## GitHub Pages

This repo is ready for GitHub Pages. For a user site, publish it as
`setliu22.github.io` and serve from the `main` branch root.
